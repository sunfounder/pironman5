OLEDスクリーン
===================

.. image:: img/oled_screen.png
    :width: 400
    

* **サイズ**: 0.96インチ
* **素材**: PM OLED
* **カラー**: 白色光
* **ドライバー**: SSD1306
* **電圧**: 3.3V
* **解像度**: 128*64
* **表示領域**: 21.74x10.86mm
* **パネルサイズ**: 26.70x19.26x1.42mm
* **ピッチサイズ**: 0.17x0.17mm
* **ピクセルサイズ**: 0.154x0.154mm
* **視野角**: 全視野
* **動作温度**: -20~70°C
* **通信方式**: IIC/SPI/パラレル
* **接続方式**: 0.5mmピッチプラグイン型FPC
