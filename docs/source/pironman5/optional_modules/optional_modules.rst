7. 可选模块：摄像头、3.5英寸屏幕、RTL-SDR
==============================================================

.. note::

    Pironman 5 系列产品不包含以下模块。  
    你需要自行准备，或在我们的官方网站购买：

    * `摄像头模块 <https://www.sunfounder.com/products/ov5647-camera-module>`_
    * `3.5英寸触摸屏 <https://www.sunfounder.com/products/touchscreen-02>`_
    * `RTL-SDR Blog V4 <https://www.sunfounder.com/products/rtl-sdr-blog-v4>`_
    
本章节将介绍如何通过集成可选的硬件模块来扩展 Pironman 5 系列的功能，包括摄像头模块、3.5 英寸触摸屏显示器，以及 RTL-SDR 无线电接收器。  
通过组合这些扩展模块，你可以增强交互性，实现实时视觉输入，并解锁高级无线信号接收功能。
