安装 Ubuntu/Kali Linux/Homebridge/Home Assistant
=========================================================================

您可以根据手头是否有 Micro SD 卡或 NVMe SSD 来选择安装方式。

**仅使用 Micro SD 卡**

  如果您使用的是 Micro SD 卡，只需按照下方的第一种方法进行操作即可。

**使用 M.2 NVMe SSD**

  * 如果您使用的是 M.2 NVMe SSD，还需要一个 **M.2 NVMe SSD 硬盘盒适配器**，通过它将 SSD 连接到计算机，然后按照第二种方法安装操作系统。
