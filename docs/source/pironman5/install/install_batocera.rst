安装 Batocera Linux
======================================================

|link_batocera| 是一款开源且完全免费的复古游戏发行版，可复制到 USB 闪存驱动器或 SD 卡中，用于将任意计算机或微型计算机临时或永久变身为游戏主机。

您可以根据手头拥有的是 Micro SD 卡还是 NVMe SSD，选择相应的安装方式。

**仅使用 Micro SD 卡**

  如果您使用的是 Micro SD 卡，只需按照下方的第一种方法操作即可。

**使用 M.2 NVMe SSD**

  * 如果您使用的是 M.2 NVMe SSD，还需要一个 **M.2 NVMe SSD 硬盘盒适配器**，用于将 SSD 连接至计算机，并按照第二种方法安装系统。
