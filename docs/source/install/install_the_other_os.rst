安装Ubuntu/Kali Linux/Homebridge/Home Assistant
=========================================================

你可以根据手头是否有Micro SD卡或NVMe SSD来选择安装方式。

**仅使用Micro SD卡**

  如果你使用的是Micro SD卡，可以直接按照下面的第一种方法进行安装。

**使用M.2 NVMe SSD**

  * 如果你使用的是M.2 NVMe SSD，你还需要一个 **M.2 NVMe SSD外接适配器**，将SSD连接到电脑，然后按照第二种方法安装操作系统。
