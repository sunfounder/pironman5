3. オペレーティングシステムのインストール
==========================================

この章では、オペレーティングシステムのインストール方法を学びます。

.. note::

    Raspberry Pi 5をサポートするOSをインストールする必要があります。最新のRaspberry Pi Imagerツールを使用してシステムをインストールしてください。現在テスト済みのシステムは以下の通りです：

    .. image:: img/compitable_os.png


1. OSをMicro SDカードにインストールする
---------------------------------------------------

Micro SDカードを使用する場合は、以下のチュートリアルに従ってシステムをMicro SDカードにインストールできます。

    .. toctree::
        :maxdepth: 1

        install_to_sd

2. OSをNVMe SSDにインストール
--------------------------------------------

NVMe SSDを使用し、システムインストールのためにNVMe SSDをコンピュータに接続するアダプタがある場合は、以下のチュートリアルを使用して迅速にインストールできます。

    .. toctree::
        :maxdepth: 1

        install_to_nvme

3. OSをMicro SDカードからNVMe SSDにコピーする
---------------------------------------------------------------

.. note::

    この第三の方法は、 **Raspberry Pi OS** のみに適用されます。別のシステムを使用している場合は、Micro SDカードからNVMe SSDにシステムをコピーする方法について、リソースを参照する必要があります。

NVMe SSDをお持ちで、NVMeをコンピュータに接続するアダプタがない場合、まずシステムをMicro SDカードにインストールできます。Pironman 5が正常に起動した後、Micro SDカードからNVMe SSDにシステムをコピーできます。詳細な手順は以下の通りです：

以下の手順に従ってください：

1. Micro SDカードにシステムをインストールします。

* :ref:`install_os_sd`

2. Raspberry Piを起動してログインします。

* :ref:`login_rpi`

3. システムをMicro SDカードからNVMe SSDにコピーするか、Raspberry Piシステムのイメージャーを使用してシステムを直接NVMe SSDにインストールし、NVMe SSDからのブートを設定します。

* :ref:`boot_from_ssd`
