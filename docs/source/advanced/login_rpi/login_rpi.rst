Raspberry Pi OSへのログイン
=====================================

この章では、Raspberry Piへのログイン方法を学びます。画面が接続されている場合でも、リモートでアクセスする必要がある場合でも、このセクションではターミナルを開く方法を案内します。ターミナルは、後の章でコマンドを入力するために使用します。

.. note::

    すでにRaspberry Piの操作に慣れている場合は、この章をスキップしても構いません。

画面を使ってログインする
---------------------------

Raspberry Piに画面を接続することで、システムと直接対話することが容易になります。

**必要なコンポーネント**

* Pironman 5
* 電源アダプター
* Raspberry Pi OSがプリインストールされたMicro SDカードまたはNVMe SSD
* モニター用電源アダプター
* HDMIケーブル
* モニター
* マウス
* キーボード

**手順**

#. Micro SDカードをPironman 5に挿入します。

#. マウスとキーボードをPironman 5のUSBポートに接続します。

#. HDMIケーブルを使用してモニターをPironman 5のHDMIポートに接続します。モニターが電源に接続され、電源が入っていることを確認します。

#. 電源アダプターを使用してPironman 5の電源を入れます。しばらくすると、モニターにRaspberry Pi OSのデスクトップが表示されるはずです。

    .. image:: img/bookwarm.png
        :align: center

#. デスクトップが表示されたら、ターミナルアイコンをクリックするか、メニューで検索してターミナルを開き、コマンドを入力します。

画面なしでリモートログインする
------------------------------------

モニターにアクセスできない場合でも、リモートでRaspberry Piを使用できます。

コマンドラインアクセスには、SSHを使用してRaspberry PiのBashシェル（デフォルトのLinuxシェル）に接続できます。これにより、コマンドを介してデバイスを管理できます。

グラフィカルインターフェイスを好む場合は、VNC Viewerのようなリモートデスクトップアプリケーションを使用することで、視覚的にファイルや操作をリモートで管理できます。

**必要なコンポーネント**

* Pironman 5 
* 電源アダプター
* Raspberry Pi OSがプリインストールされたMicro SDカードまたはNVMe SSD

**手順**

#. Micro SDカードをPironman 5に挿入します。

#. 電源アダプターを使用してPironman 5を電源に接続します。

#. 使用するコンピュータのオペレーティングシステムに応じたリモートアクセスの設定に関する詳細なチュートリアルについては、以下のセクションを参照してください。
