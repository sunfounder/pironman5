|link_pironman5| - NVMe M.2 SSD PCIe ミニPCケース（Raspberry Pi 5対応）
================================================================================

この度は、|link_pironman5| をお選びいただきありがとうございます。

.. note::
    本書は以下の言語でご利用いただけます。

        * |link_german_tutorials|
        * |link_jp_tutorials|
        * |link_en_tutorials|
    
    ご希望の言語でドキュメントにアクセスするには、該当のリンクをクリックしてください。

.. image:: img/pironman5.JPG
    :width: 400
    

Raspberry Pi 5 をスタイリッシュかつ高性能にアップグレードする |link_pironman5| 。美しさと機能性が絶妙に融合した、最高級のケースです。

* Raspberry Pi 5 ミニPC
* 0.96インチOLEDディスプレイで、Raspberry PiのCPU使用率、温度、ディスク使用率、IPアドレス、RAM使用率などを表示
* タワークーラーが、室温25°Cで100% CPU負荷のPiを39°Cまで冷却
* 2つのRGBファンがGPIOで制御される
* タワークーラー上の1つのPWMファンがRaspberry Piシステムにより制御される
* 2230、2242、2260、2280の4種類の(PCIe Gen 2.0 / PCIe Gen 3.0) NVMe M.2 SSDサイズに対応
* 4つのWS2812アドレス指定可能なRGB LEDがケース全体をクールなライト効果で照らす
* IR受信機でマルチメディアセンター（KodiやVolumioなど）に対応
* レトロメタルパワーボタンで安全にシャットダウン
* ピン名ラベル付き外部GPIO拡張子で簡単にアクセス可能
* スプリング式ソケットを備え、カードの取り外しが簡単
* クリアアクリル製サイドパネル付きアルミニウムボディ
* 標準HDMIポートを2つ搭載

|link_pironman5| は単なるケースではなく、Raspberry Pi 5 をパワフルで効率的、かつスタイリッシュなデバイスへと変貌させるアップグレードです。今すぐ手に入れて、Raspberry Piケースの未来を体験してください！
