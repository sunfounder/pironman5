Raspberry Pi OS のインストール
================================================================================

Micro SD カードまたは NVMe SSD のどちらをお持ちかに応じて、インストール方法を選択できます。

* NVMe SSD に直接インストールする場合、Micro SD カードへのインストールに比べて追加の手順が必要です。Raspberry Pi はデフォルトで Micro SD から起動するよう設定されているため、NVMe SSD からの起動を優先するようにブートローダーを更新する必要があります。
* NVMe SSD はあるがコンピュータに接続するアダプターがない場合は、まず Micro SD カードにシステムをインストールし、Pironman 5 の起動後にシステムを Micro SD から NVMe SSD にコピーするという第三の方法も検討できます。
