Ubuntu / Kali Linux / Homebridge / Home Assistant のインストール
=========================================================================

お手元にある Micro SD カードまたは NVMe SSD に応じて、インストール方法を選択できます。

**Micro SD カードのみを使用する場合**

  Micro SD カードを使用する場合は、以下の最初の方法に従ってください。

**M.2 NVMe SSD を使用する場合**

  * M.2 NVMe SSD を使用する場合は、SSD をコンピューターに接続するための **M.2 NVMe SSD エンクロージャーアダプター** が必要です。アダプターを使用して SSD を接続し、2 番目の方法に従って OS をインストールしてください。
