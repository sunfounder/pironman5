2. 组装说明
=============================================

在组装 Pironman 之前，请先确认所有零部件是否齐全。如发现有缺失或损坏，请立即通过 service@sunfounder.com 联系 SunFounder，以尽快解决问题。

**提示**

* 建议按照纸质安装说明逐步操作。
* 安装前先分类并摆放好所有组件，方便快速查找。
* 反复检查 FFC 与 FPC 排线的连接，避免接触不良。
