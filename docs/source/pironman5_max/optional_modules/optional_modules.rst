7. オプションモジュール: カメラ、3.5インチスクリーン、RTL-SDR
==============================================================

.. note::

    Pironman 5 シリーズ製品には以下のモジュールは含まれていません。  
    ご自身で用意するか、公式サイトから購入してください:

    * `カメラモジュール <https://www.sunfounder.com/products/ov5647-camera-module>`_
    * `3.5インチ タッチスクリーン <https://www.sunfounder.com/products/touchscreen-02>`_

このセクションでは、カメラモジュール、3.5インチタッチスクリーンディスプレイ、RTL-SDR ラジオ受信機といったオプションハードウェアモジュールを統合することで、Pironman 5 シリーズの機能を拡張する方法を説明します。  
これらのアドオンを組み合わせることで、インタラクティブ性を高め、リアルタイムの映像入力を可能にし、高度な無線信号受信を実現できます。
