4. ソフトウェアのセットアップまたはインストール
================================================

システムをMicro SDカードまたはNVMe SSDに書き込んだら、それをPironman 5本体に挿入し、電源ボタンを押して起動します。

起動すると、各種電源LEDが点灯しますが、この時点ではOLEDディスプレイ、RGB LED、両側のRGBファンはまだ動作しません。これらの機能は追加設定が必要です。起動直後に画面が乱れる場合がありますが、設定完了後に解消されるため、現段階では問題ありません。

設定を始めるには、まずRaspberry Piを起動し、システムにログインする必要があります。ログイン方法が不明な場合は、公式Raspberry Piウェブサイト（ |link_rpi_get_start| ）をご参照ください。

その後、ご使用のシステムに対応したセットアップガイドを選択してください。

**電源ボタンについて**

この電源ボタンは、Raspberry Pi 5の電源ボタンと同様に機能します。

* **シャットダウン**

    * Raspberry Pi **Bookworm Desktop** を使用している場合は、電源ボタンを素早く2回押すことでシャットダウンできます。
    * Raspberry Pi **Bookworm Lite** を使用している場合は、1回押すことでシャットダウンが開始されます。
    * 強制シャットダウンが必要な場合は、電源ボタンを長押ししてください。

* **電源オン**

    * Raspberry Piがシャットダウン状態でも電源が供給されていれば、電源ボタンを1回押すことで起動できます。

* シャットダウンボタンに対応していないシステムでは、電源ボタンを5秒間長押しすることで強制的に電源を切ることができ、再び1回押すことで起動します。
